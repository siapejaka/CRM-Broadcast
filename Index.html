Import Kontak dari Excel / CSV

Unggah file Excel Anda dan petakan kolomnya secara fleksibel.

Cara Ekspor Manual dari Google Contacts:

  1. Buka halaman Google Contacts (contacts.google.com)
  2. Di menu sebelah kiri, klik tombol Ekspor (Export).
  3. Pilih kontak yang ingin diekspor, lalu pilih format Google CSV.
  4. Klik Ekspor untuk mengunduh file contacts.csv.
  5. Unggah file tersebut pada area dropzone di bawah ini, lalu klik tombol "Gunakan Preset Google Contacts".
